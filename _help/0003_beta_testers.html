--- name: Beta testers ---

New versions of {{ site.github.project_title }} are released to beta testers a few days before they are made available to everyone. This allows us to have early feedbacks about the new features and make sure that we don't break anything.

To be part of the beta program of {{ site.github.project_title }}, you just have to subscribe from this link. New versions are announced in the dedicated Google+ Community.

If you want to report a bug in {{ site.github.project_title }}, you can do it in the Google+ Community, or submit an issue on GitHub.

You can unsubscribe at any time by using the same link as for the subscription. However, you might need to reinstall {{ site.github.project_title }} to have the latest stable version.
